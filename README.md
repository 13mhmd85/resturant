FoodLand Restaurant Website | Django Full Project
📘 English Version
FoodLand is a complete restaurant reservation & product showcase website built with Django, featuring:

Home Page
Product Details
User Profile Page
Edit Profile
Tags Page
Table Reservation System
Admin panel for adding products with images
Authentication (Register/Login)
Fully responsive UI with HTML, CSS, JavaScript
